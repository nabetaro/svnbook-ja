Version Control with Subversion

Someday, this will be the home of Version Control with Subversion, a free book about Subversion, a new version control system designed to supplant CVS. This book is a work in progress, which will be published by O'Reilly and Associates in mid-2004.

For now, it is simply a place to read the on-line HTML and PDF versions of the book.

Errata Note:

The book is mostly complete, but still lacking coverage on certain topics. Other topics need to be clarified. The authors have some running "to-do" lists which you can see here and here. Before emailing feedback about errors or missing information, please take a look at those lists. We may already be aware of the problem. Thanks!
