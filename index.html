Subversion: The Definitive Guide

Someday, this will be the home of Subversion: The Definitive Guide, a free book about Subversion, a new version control system designed to supplant CVS. This book is a work in progress, which will someday be published by O'Reilly and Associates.

For now, it is simply a place to read the on-line HTML version of the book. PDF and Postscript versions of the book can be found in the file-sharing section of Subversion's website.
