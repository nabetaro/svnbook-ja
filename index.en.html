Introduction

This is the online home of Version Control with Subversion, a free book about Subversion, a new version control system designed to supplant CVS. As you may have guessed from the layout of this page, this book is published by O'Reilly Media.

This is a place to read HTML and PDF versions of the book (although you can certainly buy a copy if you'd like to). We'll do our best to keep the site up-to-date. As Subversion development continues, the product will continue to grow new features, and we plan to continue documenting those changes.

Contributing

Please send any questions, comments, patches, gifts, praise, or errors related to the book itself to the Subversion development mailing list (dev@subversion.tigris.org — you need not be subscribed). Be sure to include the word [BOOK] in the subject line, and [BOOK PATCH] if submitting a patch. Please make your patches against the XML sources.

For feedback concerning this website, contact webmaster@svnbook.org.

Translations

This book has been (or is being) translated to other languages. Use the navigation menu at the bottom of the page to select a different language. From each translated page you can obtain the book or a work in progress snapshot if it is not finished yet. Note that the English version is the master from which all translations derive, and if you have any comments to do about a translation you should contact its authors. If your language is not listed here and would like to contribute a translation, follow the instructions above to contact the Subversion development mailing list and ask for details.
